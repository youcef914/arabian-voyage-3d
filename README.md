# 🌟 رحلة عبر الوطن العربي - Arabian Voyage 3D

> **استكشف جمال وثراء الوطن العربي من خلال رحلة تفاعلية ثلاثية الأبعاد**

## 📖 نظرة عامة

مشروع "رحلة عبر الوطن العربي" هو تطبيق ويب تفاعلي ثلاثي الأبعاد يتيح للمستخدمين استكشاف الدول العربية الـ22 بطريقة ممتعة وتعليمية. يجمع التطبيق بين التقنيات الحديثة والتصميم العربي الأصيل لتقديم تجربة فريدة.

### ✨ الميزات الرئيسية

- 🗺️ **خريطة ثلاثية الأبعاد تفاعلية** للوطن العربي
- 🎨 **تصميم عربي أنيق** بالألوان الكحلية والذهبية
- 🏛️ **معالم ثلاثية الأبعاد** للدول العربية
- 🎤 **مساعد صوتي ذكي** باللغة العربية
- 📱 **تصميم متجاوب** يعمل على جميع الأجهزة
- 🌙 **تأثيرات بصرية ساحرة** وانتقالات سلسة

## 🛠️ التقنيات المستخدمة

### Frontend Framework
- **React 18** - مكتبة واجهة المستخدم
- **TypeScript** - للبرمجة الآمنة
- **Vite** - أداة البناء السريعة

### 3D Graphics
- **Three.js** - محرك الرسوميات ثلاثية الأبعاد
- **@react-three/fiber** - React wrapper لـ Three.js
- **@react-three/drei** - مكونات ثلاثية الأبعاد جاهزة

### UI & Styling
- **TailwindCSS** - إطار عمل CSS
- **shadcn/ui** - مكونات واجهة المستخدم
- **Framer Motion** - مكتبة الحركة والانتقالات

### State Management
- **Zustand** - إدارة الحالة البسيطة والفعالة

### Voice Features
- **Web Speech API** - التعرف على الكلام وتحويل النص إلى كلام

## 🚀 التثبيت والتشغيل

### المتطلبات الأساسية
- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn أو pnpm

### خطوات التثبيت

1. **استنساخ المستودع**
```bash
git clone https://github.com/your-username/arabian-voyage-3d.git
cd arabian-voyage-3d
```

2. **تثبيت التبعيات**
```bash
npm install
# أو
yarn install
# أو
pnpm install
```

3. **تشغيل التطبيق في وضع التطوير**
```bash
npm run dev
# أو
yarn dev
# أو
pnpm dev
```

4. **افتح المتصفح على** `http://localhost:8080`

## 🎮 كيفية الاستخدام

### 1. شاشة الترحيب
- اضغط على زر "ابدأ الرحلة" لدخول الخريطة التفاعلية

### 2. التفاعل مع الخريطة
- **مرر الماوس** على أي دولة لرؤية اسمها
- **انقر** على دولة لاستكشافها
- **استخدم الماوس** للتنقل والتكبير/التصغير

### 3. استكشاف الدول
- **الشريط الجانبي** يظهر معلومات مفصلة عن الدولة المختارة
- **المعالم ثلاثية الأبعاد** لكل دولة
- **المأكولات الشعبية** والثقافة المحلية

### 4. المساعد الصوتي
- اضغط على **أيقونة الميكروفون** لبدء الحديث
- قل "مرحبا" أو "السلام عليكم" للترحيب
- **اسأل عن أي دولة** لسماع معلومات عنها
- اضغط على **أيقونة السماعة** لإيقاف الصوت
